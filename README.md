# SAAS-Restaurant-Management

Sistema SaaS para la gestión integral de restaurantes. API RESTful desarrollada con Spring Boot como Trabajo Práctico Final de Programación III (Comisión 7, UTN).

## Descripción

Sistema multi-tenant que permite a múltiples restaurantes registrarse, elegir un plan de suscripción y gestionar sus operaciones diarias: menú, mesas, pedidos, empleados y reservas. Incluye autenticación JWT por roles, integración con Mercado Pago y reportes de gestión.

## Stack Tecnológico

- Java 17+ / Spring Boot 3.x
- Spring Web, Spring Data JPA, Spring Security
- MySQL
- JWT (jjwt)
- Swagger / OpenAPI (springdoc-openapi)
- Mercado Pago SDK
- Maven
